# 로드맵

## 1단계

- public proto 저장소 분리
- 메인 저장소에서 proto submodule 연결
- PC 서버/클라이언트 CLI 구현
- SSDP 탐색과 gRPC 파일 송수신 구현
- 앱 레벨 파일 암호화 구현
- Flutter 클라이언트 뼈대 구성

## 2단계

- Flutter SDK 환경에서 실제 모바일 UI 구현
- QR 또는 PIN 기반 페어링 UX
- gRPC TLS와 인증서 핀닝 옵션
- 전송 진행률, 취소, 재시도
- 수신 승인 정책

## 3단계

- Windows/macOS/Linux 패키징
- Android/iOS 권한 처리
- 대용량 파일 resume
- 폴더 전송
- 기기별 신뢰 목록과 차단 목록
